# 프로젝트 기획
자신이 운동한 내용, 식단을 기록하고 이를 바탕으로 자신의 신체 변화 목표를 이뤄나가는 과정을 기록하는 서비스

# 프로젝트 관련 문서 정리
`docs/design-docs/`는 설계와 관련된 문서를 포함한다.
`docs/product-specs/`는 서비스의 수용 가능한 기준을 설립하는 내용을 포함한 문서를 저장한다.
`docs/references/`는 외부 레퍼런스를 포함한 프로젝트에 필요한 외부 문서, 내용 등을 저장한다.
`docs/exec-plans/`는 프로젝트의 실행 계획과 관련된 내용을 포함한 문서를 저장한다.

# 주의사항
답변은 무조건 한글로 생성한다. 단, 기술명, 서비스명 등 영어로 표기해도 무관한 것만 영어로 작성
